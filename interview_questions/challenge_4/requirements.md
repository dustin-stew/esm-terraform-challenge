# Terraform AWS Infrastructure Challenge

### Use of AI
Use of AI is permitted so long as its used as a tool and the candidate is able to explain their decisions.

## Scenario

You are tasked with designing AWS infrastructure using Terraform for a simple NFL scoreboard application with the following requirements:

## Application Stack

### Frontend
- Static website that displays live game scores and team standings

### Backend
- A serverless API that returns game scores and team standings

## Deliverables

### Provide terraform code that includes all the services and resources needed to meet the requirements.

> The terraform code should pass terraform plan and terraform apply without any errors.

### Data Storage
- Store game scores
- Store team standings

## Deliverables

Provide a high-level architecture description including:
- AWS services you would use and why
- How these services interact

Be prepared to discuss trade-offs in your design decisions.
